VI- LES ECRANS

1- Ecran Paramètres


Cet écran permet de rentrer les listes de choix nécessaires à la saisie des caractéristiques des zones.

POUR AJOUTER UN ITEM:

Taper le contenu de l'item dans le champ Nouveau.
  Cliquer sur le bouton AJOUTER.
POUR  REMETTRE A ZERO LE CHAMP NOUVEAU:
Cliquer sur le bouton ANNULER.
POUR SUPPRIMER UN OU PLUSIEURS ITEMS:
Sélectionner une ou plusieurs lignes dans la liste.
Cliquer sur le bouton ENLEVER.
