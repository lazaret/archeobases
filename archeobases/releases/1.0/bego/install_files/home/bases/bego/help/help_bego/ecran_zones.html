VI- LES ECRANS

2- L'écran Zones


Cet écran permet de saisir toutes les informations relatives aux zones.

AVANT LA SAISIE DE L'IDENTIFIANT: Le seul champ obligatoire pour la saisie est le champ Zone qui est la clef primaire de la table Zone.

APRES AVOIR CLIQUE SUR CHERCHER OU CREER:
 
 

 
Les liens en bleu permettent de se rendre directement sur les écrans correspondants à cette zone ainsi que sur la zone suivante.

Le tableau contenant la liste des roches remarquables est généré automatiquement à partir du champ remarquable présent dans l'écran Roche. Pour se rendre directement sur cette roche, il suffit de cliquer sur le numéro de la roche ( ici 0).

Pour modifier la liste des paramètres, il faut cliquer sur le bouton MODIFIER.
 

L'ECRAN DE MODIFICATION DES PARAMETRES: Lorsque l'on a cliqué sur le bouton MODIFIER, on aboutit alors sur l'écran suivant: (les listes des items possibles ont été précédemment saisis dans l'écran zone).

Pour ajouter un ou plusieurs accès à la zone:
Sélectionner les items à l'aide de la souris.
Cliquer sur le bouton Ajouter.
Pour enlever un ou plusieurs accès à la zone:
Sélectionner les items à l'aide de la souris.
Cliquer sur le bouton Enlever.
Pour retourner à l'écran de saisie des zones: Cliquer sur le bouton Fin Modif
