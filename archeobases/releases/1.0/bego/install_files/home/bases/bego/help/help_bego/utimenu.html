V- PRINCIPES GENERAUX D'UTILISATION

2- Le menu principal


Il permet grâce à des boutons de se rendre soit sur les écrans de consultation/saisie/modif soit sur les écrans de service.

Les écrans de consultation/saisie/modification permettent de remplir les tables Zone, Roche, Face, Figure, Historique et Association.

L'écran "Paramètres"  permet de renseigner un certain nombre de paramètres relatifs aux zones.

L'écran "SQL" donne accès à une interface directe avec la base de données grâce au langage de requête SQL

Le lien "Album" enverra dans un avenir proche vers une photothèque des documents en relation avec la base de données du Bego ( photographies, diapositives, calques, cheminements)

  Le lien "Présentation" renvoie comme son nom l'indique vers une Présentation de l'application Mont Bego ainsi qu'une aide pour l'utilisation de celle-ci.
