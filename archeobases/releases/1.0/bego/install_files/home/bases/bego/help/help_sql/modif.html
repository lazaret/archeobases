2- MODIFIER LA BASE DE DONNEES


A- Insertion

L' ajout de lignes dans une table se fait par la commande INSERT INTO.
 

INSERT INTO nom_table ( nom_champ1, nom_champ2, ...)
VALUES ( valeur_champ1, valeur_champ2, ...);


Si l'on ne précise pas les nom_champ, par défaut la liste des nom_champ sera la liste complète des champ de la table. Par contre dans ce cas là, il faut bien donner une valeur pour tous les champ de la table dans le VALUES et dans le bon ordre ( donc fortement déconseillé).

INSERT INTO nom_table
VALUES ( valeur_champ1, valeur_champ2, valeur_champ3)
B- Modification

La modification se fait sur un ou plusieurs champ avec le mot clef UPDATE.
 

UPDATE nom_table
SET nom_champ1 = nouvelle_valeur1,
          nom_champ2 = nouvelle_valeur2,
          nom_champ3 = nouvelle_valeur3
WHERE condition;


C- Suppression

Pour supprimer une ou plusieurs lignes d'une table, on utilise la commande DELETE.

DELETE FROM nom_table
WHERE condition;
