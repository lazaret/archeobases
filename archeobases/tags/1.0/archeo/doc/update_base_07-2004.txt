Avant modif
===========
faire un pg_dumpall > sauvegarde_totale
faire pour chaque base un pg_dump -C -d nom_base > sauvegarde_nom_base
copier ces sauvegardes sur CD et sur une autre machine
mettre de côté la bande de sauvegarde de la veille après s'être assuré qu'elle a bien fonctionné.


Pour effectuer les modifs copier coller les commandes apres avoir tapé psql nom_base .
Les exécuter une par une et vérifier systématiquement que la commande s'est bien déroulée avant de passer à la suivante sous peine d'être obligé de restaurer la base concernée.

Ajout du champ ordre de la table nucleus:
========================================
DROP INDEX nucleus_pkey;

CREATE TABLE "nouv_nucleus" (
"zone" text NOT NULL,
"numero" integer NOT NULL,
"bis" text NOT NULL,
"n_ordre" integer NOT NULL,
"n_type" text,
"n_nbplan" integer,
"n_nba" integer,
"n_nbb" integer,
"n_nbp" integer,
"n_orientation" text,
"n_dirtotal" integer,
"n_dirfacea" integer,
"n_dirfaceb" integer,
"n_dirprofil" integer,
"n_formule" text,
"n_rotation" text,
"n_cortotal" integer,
"n_corfacea" integer,
"n_corfaceb" integer,
"n_corprofil" integer,

"n_epuisement" integer,
Constraint "nucleus_pkey" Primary Key ("zone", "numero", "bis", "n_ordre"), FOREIGN KEY (zone, numero, bis) REFERENCES industrie (zone, numero, bis)
);

INSERT INTO nouv_nucleus 
SELECT zone, numero, bis, 1, n_type, n_nbplan,n_nba,n_nbb,n_nbp,n_orientation,n_dirtotal,n_dirfacea,n_dirfaceb,n_dirprofil,n_formule,n_rotation,n_cortotal,n_corfacea,n_corfaceb,n_corprofil,n_epuisement

DROP TABLE nucleus;

ALTER TABLE nouv_nucleus
RENAME TO nucleus;
