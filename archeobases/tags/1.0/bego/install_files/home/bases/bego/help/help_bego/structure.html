LA TABLE REQUETE

C'est la table qui permet de stocker les requêtes avec un nom, afin de pouvoir s'en resservir. En pratique, il n'y a pas besoin de l'interroger directement. Il suffit de cliquer sur la requête désirée dans l'écran SQL.
