UTILISATION DE L'APPLICATION
5- L'écran SQL


 


Il permet d'effectuer des requêtes SQL pour interroger la base de données.
 


 

Saisir la requête dans la zone de texte. Eventuellement se servir du bouton Aide

Saisir le Nom de la requête

Choisir le type d'affichage que l'on désire. Il ya quatre options possibles:

Simplifié                => L'affichage s'effectue en colonnes.

Elaboré                  => L'affichage du résultat se fait dans un tableau.

Texte + Tabs       => L'affichage du résultat se fait dans un fichier et les colones de résultat sont séparées par des tabulations.

Texte + Virgules =>L'affichage du résultat se fait dans un fichier et les colones de résultat sont séparées par des virgules.

Le principal intérêt d'envoyer le résultat dans un fichier est que l'on peut ensuite récupérer le résultat depuis un traitement de texte ou bien un tableur. Le résultat peut ensuite être aisément mis sou forme de tableau en remplaçant ce qui est séparé par les espaces ou bien les tabulations par cellules de tableau.
Eventuellement cliquer sur ANNULER si la requête ne convient pas.

Ensuite cliquer sur le bouton ENVOYER.
Si on a choisi d'envoyer le résultat dans un fichier, la fenêtre suivante va s'ouvrir pour permettre de choisir le nom du fichier dans lequel on veut enregistrer le résultat de la requête.


 


Le lien Aide envoie vers une iniation au langage de requête avec des exercices corrigés.

  Le lien Liste des requete affiche dans un tableau toutes les requêtes qui ont été sauvegardées. On sauvegarde une requête tout simplement en lui donnant un nom. Il suffit ensuite pour lancer la requête de cliquant sur celle-ci dans le tableau. Il peut être utile de lancer cette liste pour se servir des requêtes déjà effectuées comme modèle.
