TEXTE DES EXERCICES

Soit une table stagiaire comprenant 6 champs:

=> id_stagiaire             ( champ numérique)
=> nom_stagiaire        ( champ texte)
=> prenom_stagiaire  (champ texte)
=> note1                         (champ numérique)
=> note2                         (champ numérique)
=> id_option                  ( champ numérique)


soit une table option comprenant 3 champs:

=> id_option (champ numérique)
=> nom_option (champ texte)
=> coefficient (champ numérique)
1- INTERROGER LA BASE

A- Sélectionner les données:

1- Editer toutes les informations contenues dans la table option
2- Editer toutes les informations contenues dans la table stagiaire
3- Editer uniquement l'identifiant et le nom des stagiaires
4- Chercher tous les prénoms représentés par les stagiaires
5- Chercher toutes le informations concernant le stagiaire vaudron:
6- Editer le nom des stagiaires qui ont leur première note supérieure ou égale à 5 et leur deuxième note inférieure à 7
7- Editer l'identifiant et le nom des stagiaires qui ont leur note1 comprise entre 3 et 8
8- Editer les noms des stagiaires dont les prénoms sont maxime ou karine
9- Editer toutes les informations concernant les stagiaires dont les noms commencent par du
10- Editer les prénoms des stagiaires dont la deuxième lettre du prénom est un a et dont le prénom comporte 6 lettres
11- Editer toutes les informations concernant les options dont le coeff a été saisi
12- Afficher le nom et le prenom des stagiaires suivis de l'option qu'ils ont choisie et du coefficient correspondant à cette option
13- Afficher toutes les informations concernant les stagiaires qui ont choisi une option qui a été saisie

B- Ordonner les données:

1-  Afficher les noms et prénoms des stagiaires par ordre alphabétique
2- Etablir le classement de la meilleure à la plus mauvaise note, en affichant le nom du stagiaire correspondant
3- Afficher dans une colonne le nom des stagiaires et dans une autre le total de leur deux notes par odre croissant

C- Les opérateurs ensemblistes
1- Afficher le nombre des stagiaires par id_option
2- la clause HAVING  permet de préciser la condition à laquelle le GROUP BY doit s'exécuter. Elle fonctionne de la même façon que la clause WHERE.

D- Les fonctions:
1- Afficher les moyennes des stagiaires
2- Afficher le nom de tous les stagiaires en majuscules et leur prénom avec  seulement la première lettre en majuscules

E- Les fonctions de groupe
1- Calculer la moyenne des note1
2- Compter le nombre de stagiaires
3- Afficher la plus mauvaise note1, et la meilleure note2
4- Calculer la somme des note1
5- Calculer l'écart-type des note2
7- Calculer la variance des note2
 

2- MODIFIER LA BASE DE DONNEES

A- Insertion
1- Insérer dans la table  option une option SQL ayant pour identifiant 3.
2- ajouter une option programmation ayant pour identifiant 4 et pour coefficient 4.

B- Modification
1- Mettre le coefficient de l'option bureautique à 2.
2- Mettre le coefficient à 3 partout où il est à 4 et partout où il est à 2.

C- Suppression
1- Supprimer l'option informatique de la table option
